Newbee-API

https://api-newbee.herokuapp.com/ #### API implementation using Flask and SQLite with 3 main endpoints for student and subject managements as well as a directory for college departments

# Student endpoint ttps://api-newbee.herokuapp.com/student ##### GET method On a GET method this endpoint returns a JSON with all the registered students ##### POST method n a POST method the API will insert a student object to the database following the models describe at the end of this file https://api-newbee.herokuapp.com/student/matricula ##### GET method On a GET method this endpoint will return a JSON with all the details of the student object that has the specified "matricula" number. ##### PUT method On a PUT method the endpoint will update the specified student that matches the given "matricula". ##### DELETE method On a DELETE method this endpoint will delete from the database the student with the specified "matricula" and return the information from the deleted object.
